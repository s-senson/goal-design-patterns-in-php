goal-design-patterns-in-php
===========================
These are simple examples of the following design patterns:

1. Adapter
  - allows interface of one class to be used from another interface
2. Factory
  - uses factory methods to deal with the creating objects
3. Singleton
  - restricts instantiation of one class to one object
4. Strategy
  - enables an algorithm's behavior to be selected at runtime 
5. Template
  - defines the program skeleton of an algorithm
